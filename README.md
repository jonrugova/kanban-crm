# Kanban CRM – Client Onboarding Prototype

## Overview

This project is a Kanban-style CRM prototype built for a tax advisory firm's client onboarding process. It is designed to provide a simple, intuitive workflow for managing clients from the initial inquiry through becoming an active client.

## Features

* Drag and drop client cards between onboarding stages
* Create new clients directly in any column
* View, edit and delete client records
* Stage-specific onboarding details
* Search clients by name or email
* Client detail modal
* Persistent data using localStorage
* Responsive, clean interface inspired by the company's branding

## Tech Stack

* React
* Vite
* TypeScript
* CSS
* dnd-kit
* localStorage

## Why This Stack

I chose React and Vite to rapidly build a modern, component-based interface. TypeScript improves maintainability and type safety, while localStorage keeps the application simple without introducing unnecessary backend complexity for this prototype.

## Running Locally

```bash
npm install
npm run dev
```

## Development Approach

I started by designing the onboarding workflow and board structure, then implemented drag-and-drop functionality, client management, persistence, search, and stage-specific forms. The focus throughout was usability, clarity, and keeping the scope aligned with the assignment.

## AI Usage

AI was used to assist with brainstorming, component implementation, debugging, and code review. All architectural decisions, feature selection, testing, and refinement were made manually.
